## 开发

```bash
$ pnpm i
$ pnpm dev
```

## 技术栈

- 语言：TypeScript
- 框架：Nextjs
- 测试：Vitest、@testing-library/react
- 构建工具：Webpack
- 包管理器：pnpm
- 样式：Less、Less Module
- UI：Arco Design
- 状态管理：Mobx
- 代码风格：Eslint、Prettier
- 代码提交：Husky、Commitlint、lint-staged、cz-git、bump-version
